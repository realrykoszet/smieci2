## SEKCJA 1: CZYM JEST "FAKT + PUENTA"

### Wzorzec Lukasza (genialny przyklad):

> "Wyobraz sobie ze rekin zyje dluzej niz niektore panstwa. Taki sobie spokojnie plywa, a w tym czasie na ladzie ludzie zdazyli wynalezc internet, upadek komunizmu i piec nowych diet bananowych."

### ROZBIJMY TO:
- **FAKT**: rekin zyje dluzej niz panstwa (wow, nie wiedzialem)
- **OBRAZ**: "taki sobie spokojnie plywa" (widze to w glowie)
- **KONTRAST**: spokojny rekin vs chaotyczni ludzie (napiecie)
- **PUENTA**: "piec nowych diet bananowych" (absurd -> smiech -> ZAPAMIETANIE)

### DLACZEGO TO DZIALA:
1. **Zaskoczenie** -> mozg sie budzi
2. **Obraz** -> wyobraznia pracuje
3. **Kontrast** -> napiecie rosnie
4. **Puenta z humorem** -> endorfiny -> ZAPAMIETANIE

### FORMUŁA (do powtarzania):
```
FAKT (zaskakujacy) + OBRAZ (widzisz to) + KONTRAST (napiecie) + PUENTA (usmiech) = ZAPAMIETANIE
```

## SEKCJA 3: ZASADY SMAKU (kiedy mozesz sobie pozwolic)

```
KONTEKST                        → POZIOM HUMORU

LinkedIn post                   → PELNY smak (analogie, puenty, self-deprecating)
Instagram                       → ODWAZNY (krotkie, wizualne, memowalne)
Cold email (1. kontakt do CISO) → DELIKATNY (1 analogia max, profesjonalnie)
Follow-up #2                    → SREDNI (mozna poluzowac — klient juz wie kim jestes)
Follow-up #3+                   → PELNY (i tak nie odpowiedzial — nie masz co stracic)
Spotkanie 1:1                   → NATURALNY (reaguj na sytuacje, nie recytuj zarty)
Oferta PDF                      → ZERO (profesjonalnie, bez humoru)
Grant ECCC                      → ABSOLUTNE ZERO
Odpowiedz na reklamacje         → ZERO + empatia
Newsletter                      → SREDNI-PELNY (subskrybent juz Ci ufa)
Demo / PoC call                 → DELIKATNY (1-2 analogie, reszta merytorycznie)
Breakup email (ostatni kontakt) → PELNY + odwaga (nic do stracenia, moze sie usmiechna)
```

### ZASADA WYCZUCIA:
```
Czytasz wiadomosc klienta:
  Czy jest zrelaksowany?     → humor OK
  Czy jest zestresowany?     → empatia, potem lekki humor
  Czy jest wsciekly?         → ZERO humoru, 100% rozwiazanie
  Czy nie odpowiada?         → humor w follow-up (bo i tak nie masz co stracic)
  Czy to pierwszy kontakt?   → 1 analogia, reszta profesjonalnie
  Czy juz kupowal?           → mozesz sobie pozwolic, bo juz Ci ufa
```

### ZASADA PROPORCJI:
```
Cold email:    90% merytoryka / 10% smak (1 analogia, reszta twardo)
Follow-up:     70% merytoryka / 30% smak (wicej analogii, PS z humorem)
LinkedIn post: 40% merytoryka / 60% smak (hook + puenta = KLIKI)
Instagram:     20% merytoryka / 80% smak (obraz + caption + CTA)
Newsletter:    60% merytoryka / 40% smak (wartosc + osobowosc)
```

## SEKCJA 5: JAK TWORZYC NOWE FAKT+PUENTA (instrukcja dla Discovery Engine)

### PRZEPIS W 5 KROKACH:

```
KROK 1: FAKT
  Znajdz ZASKAKUJACY fakt (cos czego klient NIE WIE)
  Zrodla: raporty branżowe, statystyki EU, badania naukowe, ciekawostki przyrodnicze
  TEST: czy osoba unieslaby brew? TAK → dobry fakt. NIE → szukaj dalej.

KROK 2: OBRAZ
  Stworz OBRAZ (klient musi TO ZOBACZYC w glowie)
  Technika: uzyj czasownikow akcji, kolorow, zmyslów
  ZLE: "Haker uzyskal dostep do systemu"
  DOBRZE: "Haker taki sobie spokojnie siedzi w Twoim serwerowni od 204 dni"

KROK 3: KONTRAST
  Zbuduj KONTRAST (duze vs male, szybkie vs wolne, drogie vs tanie)
  Wzorzec: [cos wielkiego/powaznego] vs [cos malego/absurdalnego]
  Przyklad: "6.8 miliarda EUR kar" vs "Andora moglaby kupic sie dwa razy"

KROK 4: PUENTA
  PUENTA z usmiechem (absurd, codziennosc, zaskoczenie)
  Techniki:
    - ABSURD: doprowadz logike do absurdu ("...i piec nowych diet bananowych")
    - CODZIENNOSC: porownaj z czyms co kazdy zna ("jak SMS od mamy")
    - SELF-DEPRECATING: posmiej sie z siebie ("firme latwiej niz zmywarke")
    - TWIST: ostatnie slowo zmienia wszystko ("...napisane Comic Sans")

KROK 5: TEST LUKASZA
  Przeczytaj glosno. Czy Lukasz by sie usmlechnal?
    TAK → uzywaj
    NIE → wyrzuc i zacznij od kroku 1

  DODATKOWY TEST: czy mozna to powiedziec na konferencji?
    TAK i ludzie sie usmiechna → ZLOTO
    TAK ale cisza → popraw puente
    NIE bo zbyt kontrowersyjne → zmien analogie
```
